### SDE- 2 Challenge

# Build an E-Commerce Platform

Build a production-ready e-commerce platform.

#### WHAT TO BUILD

### ● Build a fully functional e-commerce application with product browsing and purchasing —

### include the features you think are necessary to make it complete and usable.

#### TECH STACK

You are free to use the following. If you want to use something not listed, go for it — just mention it
in your README.

- React.js — primary framework. Next.js is also acceptable
- JavaScript (ES6+) — TypeScript is welcome if you prefer
- CSS / SCSS — for all styling
- State management of your choice — Redux Toolkit, Zustand, Context, or anything else
- Any open-source libraries you find useful
- No backend required — mock data or free public APIs are fine

#### HOW TO SUBMIT

1. Share a public GitHub repository link or a CodeSandbox link with your recruiter.
2. Make sure the application runs with standard setup commands (npm install && npm start or
   equivalent).
3. Include a README.md — tell us what you built, the decisions you made, and anything you would
   do differently with more time.
